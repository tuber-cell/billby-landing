## 📋 What This Tool Does

- **GST calculation** — Split CGST, SGST, and IGST instantly for accurate invoicing.
- **Reverse GST** — Find the base amount from a total that already includes GST.
- **10 free invoices** — Generate professional invoices without signing up.
- **Social proof** — Live counter, activity feed, and scarcity timer to build trust.
- **Mobile-friendly** — Works perfectly on all devices.

---

## 🧾 How to Use

1.  Enter an amount in the GST calculator.
2.  Select the GST rate (5%, 12%, 18%, or 28%).
3.  See the CGST, SGST, and IGST breakdown instantly.
4.  Click "Create Invoice in Billby" to generate a free invoice.
